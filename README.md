# CreatorDNA

CreatorDNA is a browser-extension-based creator intelligence system for short-video platforms. V1 targets **Douyin Web**.

It analyzes publicly visible short-video content through a browser extension and AI pipeline, then summarizes a creator's content structure, speaking style, shooting method, and reusable creative patterns.
